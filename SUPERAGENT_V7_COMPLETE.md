## üì¶ Installation & Setup

### Prerequisites
- Python 3.11+
- Node.js 18+
- Docker
- PostgreSQL 14+

### Quick Start

```bash
# Clone repository
git clone https://github.com/superagent/superagent-v7.git
cd superagent_upgraded

# Install dependencies
pip install -r requirements.txt
npm install

# Set environment variables
cp .env.example .env

# Start development server
python -m uvicorn api.index:app --reload

# Access API
# http://localhost:8000/docs
```
